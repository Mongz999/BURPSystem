\# 🍔 BURP! Canteen POS System



A simple Point of Sale (POS) system built using \*\*C# (Windows Forms)\*\* and \*\*MS Access Database\*\*.



\---



\## 🎯 Features



\* 🔐 Login System (Admin \& Users)

\* 📝 User Registration

\* 💰 Wallet System (Balance + Top-up)

\* 🍽 Product Menu Display

\* 🛒 Add to Cart

\* ➖ Remove Items

\* 🧾 Receipt Generation

\* 🖨 Print Receipt

\* 🖼 Profile Picture Upload

\* 📊 VAT Calculation



\---



\## 🧱 Technologies Used



\* 💻 C# (Windows Forms)

\* 🗄 MS Access Database (.accdb)

\* 🛠 Visual Studio 2012

\* 🔄 Git \& GitHub



\---



\## 🧪 How to Run



1\. Download or clone this repository:



git clone https://github.com/Mongz999/BURPSystem.git



2\. Open the project in \*\*Visual Studio 2012\*\*



3\. Make sure the database file is included:



CanteenDB.accdb



4\. Run the system (Press F5)
